``` bash

```
## Description

Brain Games is a collection of arithmetic games. Includes:

- brain-even
  
  ``` bash
  make brain-even
  ```
  
- brain-calc

  ``` bash
  make brain-calc
  ```
  
- brain-gcd

  ``` bash
  make brain-gcd
  ```
  
- brain-progression

  ``` bash
  make brain-progression
  ```
  
- brain-prime

  ``` bash
  make brain-even
  ```
